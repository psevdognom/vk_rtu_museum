Инструкция по добавлению мест в каталог

Данная форма позволяет добавлять, либо обновлять места для хранения в каталоге

Перед внесением места (местом может быть этаж, комната, стеллаж, ящик, полка и тд) необходимо узнать его физическое расположение

1. В поле с названием Name вводится название добавляемого место. Данное название должно быть уникальным

2. В поле ParenId необходимо выбрать местоБъ, внутри которого будет находиться добавляемое место

3. Далее в поле с названием Description вводится описание добавляемого места при необходимости

4. После проверки всех вводимых значений, для сохранения места необходимы нажать кнопку Добавить. При возникновении ошибки необходимо снова прочесть инструкцию. При успешным добавлении вы перейдете на новую старницу
