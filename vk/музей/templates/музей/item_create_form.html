Инструкция по добавлению мест в каталог

Данная форма позволяет добавлять, либо обновлять экспонаты в каталоге

1. В поле с названием Name вводится название добавляемого экспоната. Данное название должно быть уникальным

2. В поле с названием Mark вводится шифр добавляемого экспоната. Данное название должно быть уникальным

3. В поле Image необходимо загрузить фотографию. Доступны форматы png, jpeg и другие

4. Далее в поле с названием Description вводится описание добавляемого экспоната при необходимости

5. В поле с названием Category необходимо выбрать категорию, к которой относится данный экспонат, либо выбрать Без категории

6. В поле с названием Place необходимо выбрать место в котором находится данный экспонат

7. Далее Вы можете заполнить остальные поля при наличие данных сведенй, таких как материал, способ получения, состояние

8. После проверки всех вводимых значений для сохранения места необходимы нажать кнопку Добавить Экспонат. При возникновении ошибки необходимо снова прочесть инструкцию. При успешным добавлении вы перейдете на новую старницу
