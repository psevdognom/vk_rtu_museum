Инструкция по добавлению категорий в каталог

Данная форма позволяет добавлять, либо обновлять категории для в каталоге

Экспонаты разбивается на катнгории логически, например медалии, вымпелы, внутри каждой категории может быть подкатегория

1. В поле с названием Name вводится название добавляемой категории. Данное название должно быть уникальным

2. В поле ParenId необходимо выбрать категории внутри которой будет находиться добавляемая подкаетегория. Либо категориия не будет иметь родительской категории и бужет являться одной из корневых

3. Далее в поле с названием Description вводится описание добавлямой категории при необходимости

4. После проверки всех вводимых значений для сохранения категории необходимы нажать кнопку Добавить. При возникновении ошибки необходимо снова прочесть инструкцию. При успешным добавлении вы перейдете на новую старницу
